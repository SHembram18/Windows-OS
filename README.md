### To Check the saved wifi password 👁‍🗨
- Open your command prompt as Adminstrator
- Run 2 commands
- 🖥netsh wlan show profile
- 🖥wlan export profile folder=c:\ key=clear
- Now the password will be store in your C drive in .XML extension
